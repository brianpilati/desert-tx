Installation


# Requirements 

## node v10.7.0 (or equivalent)
## npm v6.1.0 (or equivalent)
## npm install -g @angular/cli

# Installation

## npm install

# Configuration

copy the provided `serviceAccountKey.json` to the `node-server/libs` directory

# Execution

## Node Server
$ cd node-server
$ node app.js

## Angular Frontend
$ ng serve

In a browser open http://localhost:4200
